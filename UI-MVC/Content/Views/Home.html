Explore the magic algorithms

Canopy

The canopy clustering algorithm is an unsupervised pre-clustering algorithm. It is intended to speed up clustering operations on large data sets.

Cobweb

Cobweb incrementally organizes observations into a classification tree. Each node in a classification tree represents a class or also called concept.

KMeans

K-means clustering is a method of vector quantization that is popular for cluster analysis in data mining.

Self Organising Map

A self-organizing map (SOM) is a type of artificial neural network (ANN) that is trained using unsupervised learning.

XMeans

Extending K-means with Efficient Estimation of the Number of Clusters. X-Means is K-Means extended by an Improve-Structure part.

Expectation Maximization

In statistics, an expectation maximization (EM) algorithm is an iterative method for finding maximum likelihood or maximum a posterior (MAP).
