Sustainable Solvents

Sussol is a research project by two departments of the Karel de Grote Hogeschool at Antwerp, An easy to use solvent clusterer platform that is based on the artificial intelligence library WeKa, We search alternative sustainable solvents to replace existing solvents and our system shows the result with clusters, You can create your own organisation and share your results with your colleagues or friends, The project is a collaboration between the Applied Informatics department and the Chemical departments of KDG Antwerp

Explore the magic algorithms

Canopy

The canopy clustering algorithm is an unsupervised pre-clustering algorithm. It is intended to speed up clustering operations on large data sets.

KMeans

K-means clustering is a method of vector quantization that is popular for cluster analysis in data mining.

Self Organising Map

A self-organizing map (SOM) is a type of artificial neural network (ANN) that is trained using unsupervised learning.

Expectation Maximization

In statistics, an expectation maximization (EM) algorithm is an iterative method for finding maximum likelihood or maximum a posterior (MAP).

WeKa Engine

In this project we are working with the open source WeKa Machine Learning Engine. Weka is a collection of machine learning algorithms for data mining tasks and analyses. Weka contains tools for data pre-processing, classification, regression, clustering, association rules, and visualization.

Read more Weka

Producer Web Service

We implemented the WeKa library in our own Java Application and build a Webservice shell around it. Now you can access the algorithms with a few simple API-calls.

Read more Producer

Pre-processing

Data pre-processing is a very important step in the data mining process. Data pre-processing includes tasks like cleaning, normalization, selection, ... In our case we used a data matrix containing 6070 solvents with each 48 physical and/or chemical, 25 (eco) toxologic, 19 CLP regulation symbols and 204 H/P/EUH features. This data set is cleaned and transformed in the correct WeKa format.

Read more Data processing

Learning process

The result of the pre processed data is processed or learned by 4 algorithms: K-Means, Canopy, Self-Organising-Map neural network and Expectancy Maximisation.

Read more Learning process

Learning Model

The learning proces results in a model that contains cluster -and solventinformation. This model can be used to request information about the classification of the solvents.

Read more Model

Classify my own solvent

The learned model can also be used to offer and classify a new solvent. In this way the solvent will be classified, using a specific algorithm, in the most applicable cluster.
